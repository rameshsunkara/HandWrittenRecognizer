Steps to run the project:


1. This is a Maven Project.

2. Import it as Maven project into IDE. If you do not have Maven installed, please add the dependencies specified in pom.xml to your class path.

3. Main class is present in Main.java.

4. Run it as Java application (Main.java)

5. Specify the JVM memory in the run configuration as  -Xms4096m

6. You can find source code on github.
https://github.com/rameshsunkara/HandWrittenRecognizer
